# Sales Dashboard - Power BI 📊

## 📌 Overview
Created an interactive Power BI dashboard to analyze sales, profit, category, and regional performance.

## 🛠 Tools Used
- Power BI
- CSV Dataset
- Data Visualization

## 📊 Features
- Total Sales & Profit KPI Cards
- Category-wise Sales Analysis
- Region-wise Profit Distribution
- Interactive Filters

## 🚀 Outcome
Improved understanding of business intelligence and dashboard creation.
